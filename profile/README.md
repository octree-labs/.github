# Octree

**AI LaTeX Editor for Fast Scientific Writing**

Octree is an intelligent LaTeX editing platform that combines real-time PDF compilation, AI assistance, and tools to help researchers, students, and professionals create publication-ready documents efficiently.

**Website**: [Octree](https://useoctree.com)  
**Free Tools**: [Tools](https://tools.useoctree.com/)
**Free Templates**: [Templates]([https://tools.useoctree.com/](https://tools.useoctree.com/templates))

---

## Why Octree?

Traditional LaTeX editing can be complex and time-consuming. Octree transforms the experience with:

- **AI-Powered Writing** – Generate LaTeX code, fix errors, and get intelligent suggestions
- **Real-Time Compilation** – See your PDF output instantly as you type
- **Zero Setup Required** – Start writing immediately, no installation needed
- **TikZ Diagram Generation** – Create complex diagrams with AI assistance
- **Smart Document Structure** – Organize sections and references automatically

## Projects

### [Octree Editor](https://github.com/octree-labs/octree)
The main LaTeX editor application featuring:
- Real-time collaborative editing
- AI-assisted LaTeX generation and error correction
- Instant PDF preview and compilation

### [Octree Tools](https://github.com/octree-labs/tools)
Free online LaTeX conversion and utility tools:
- Markdown to LaTeX converter
- Table generators
- Equation formatters
- BibTeX management
- Symbol reference guides
- Many templates
